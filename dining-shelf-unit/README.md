## Dimensions
* 30"W x 14"D x 60"H
* Shelf(4): 30"W x 14"Dx 1.5"H
* Legs and shelf borders are 1.5" x 1.5"
* Shelves are 3/4" plywood (surrounded by hardwood border)

## Joinery
Shelf rails attach to legs with 0.5" lapped, haunched mortise & tenon
